# 更新日志

所有关于本项目值得关注的更改都将记录在此文件中。

## [2026-04-13] - 增加本地模型连接及 OMLX 支持

### 新增

- **接入本地模型**: 支持直接连接并调用部署在本地回环地址（localhost, 127.0.0.1）和私有网络（RFC 1918, .local, .internal）的 AI 模型后端。
- **OMLX 生态支持**: 全面支持整合 omlx 等基于本地/私有化部署的模型作为 Provider。

### 更改

- **放宽 URL 拦截限制**: 修改了 `internal/http/providers.go` 中的 URL 验证逻辑，取消了对私有 IP 段及内部主机名作为服务端点的统一拦截。
